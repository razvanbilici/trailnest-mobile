# TrailNest Mobile

Port React Native al aplicatiei web TrailNest

## Features

- Light / dark mode + detectie system-default 
- Autentificare mock email + password authentication 
- Navigare de tip bottom-tab
- Favorites system persisted across launches
- Search & difficulty filtering
- Integrare cu WhatsApp pentru distribuirea traseelor catre contactele utilizatorului
- Sistem tip Stories

## Tech stack

- Expo SDK 52 / React Native 0.76
- TypeScript
- React Navigation v7
- Zustand + AsyncStorage (pentr state)
- `lucide-react-native` for icons

## Getting started

```bash
cd trailnest-mobile
npm install
npx expo start
```

Scaneaza QR-ul din terminal cu aplicatia mobila **Expo Go** (VERSIUNE <= 52)
