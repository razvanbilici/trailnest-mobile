# Assets

Drop the following PNGs in this folder before publishing (they're referenced from `app.json`):

- `icon.png` (1024×1024)
- `splash.png` (1242×2436)
- `adaptive-icon.png` (1024×1024, foreground)
- `favicon.png` (48×48)

For local Expo Go development the app runs fine without them; Expo will substitute its defaults.
